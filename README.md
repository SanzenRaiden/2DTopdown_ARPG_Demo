# 项目介绍
本项目是基于 Unity 6 开发的一款 2D 俯视角战斗游戏。
采用 Unity2D 项目搭建地形场景，包含人物与怪物以及基础TileMap。

## 实现功能
1. 角色控制系统：移动、朝向控制，带有受击判定、生命值系统、近战攻击动画及伤害判定、远程攻击子弹创建及伤害判定。
2. 怪物系统：怪物具备追击范围、追击仇恨逻辑。
3. 场景设置：添加TileMap系统，检测玩家进行图层切换实现“上楼”效果，边界空气墙，限制玩家活动范围，地形碰撞完整。
4. UI界面：包含血量UI，经验值UI，角色属性面板，角色技能加点UI。
5. 使用 Git 进行版本管理，配置.gitignore过滤冗余文件。
